## Modular Configuration (Page 6)

**5 Modules Total:**
- Module 1: First floor unit (left)
- Module 2: First floor unit (right) 
- Module 3: Second floor unit (left)
- Module 4: Second floor unit (right)
- Module 5: Balcony/Deck (installed on-site)

**Structural System:**
- Floor: 6" SIP floor panels over TJI joists @ 14" O.C.
- Walls: 8" SIP wall panels
- Roof: SIP roof panels @ 24" O.C.
- Ceiling: TJI ceiling joists
- Connection: 16d common nails @ 12" O.C., panel screws @ 8" O.C.
- Rim board: 1-1/2" x 9-1/2" LVL rim joist
- Butt sections together on-site

## Key Specifications

**Exterior:**
- 7/8" three-coat stucco over metal lath + 2 layers Grade D building paper
- DEX-O-TEX Weatherwear deck covering
- Optional standing seam roof over 2 layers 30# felt
- Tyvek/StuccoWrap weather barrier system
- Drip cap/head flashing details

**Interior:**
- 5/8" Type X gypsum board at walls & ceilings
- 1/2" gypsum board each side of SIP panels
- Wood stairs with DEX-O-TEX system
- Wood guardrails with 4x4 posts

**Mechanical/Electrical:**
- RTU with factory-installed disconnect
- GFCI protected weather-resistant receptacles (extra duty)
- Max 5% voltage drop
- Water closets: 1.28 GPF
- Lavatory faucets: 1.2 GPM
- Kitchen faucets: 1.8 GPM
- Showerheads: 1.8 GPM
